# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-03-22

### Added
- Full functionalities of MPECSS solver.
- Support for MacMPEC, MPECLib, and NOSBENCH benchmark suites.
- User friendly interface for command line execution.

## [1.0.0] - 2026-03-22

### Added
- Initial release of MPECSS solver.
- Support for MacMPEC, MPECLib, and NOSBENCH benchmark suites.
- Phase I (feasibility) and Phase II (main solver) implementation.
- LPEC-based B-stationarity certification.
